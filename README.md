# todo-vue-app

## Запуск проекту через Vue CLI

1. Встановити залежності:
   ```
   npm install
   ```

2. Запустити проект у режимі розробки:
   ```
   npm run serve
   ```

3. Зібрати проект для продакшну:
   ```
   npm run build
   ```

### Lints and fixes files
```
npm run lint
```
